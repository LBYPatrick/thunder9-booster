# thunder9-booster
Travis CI测试状态:[![Build Status](https://travis-ci.org/LBYPatrick/thunder9-booster.svg?branch=master)](https://travis-ci.org/LBYPatrick/thunder9-booster)
## 简介
用一种简单暴力的方式移除迅雷9的广告，同时减少无用的资源消耗，让迅雷重新回归迅雷极速版的畅快.

## 使用方法
下载最新Release的ThunderBrowser.exe后替换迅雷根目录下TBC目录的同名文件即可.

## 一点碎碎念
自己造了很多的轮子在util.cpp里面，因为蛋疼的Windows API机制，这些轮子几乎是废的（除非你能容忍一闪而过的黑窗...），你们可以拿去玩。
